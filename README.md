# 🚀 Personal Portfolio

A modern and responsive personal portfolio website built with React, TypeScript, and Vite to showcase my projects, technical skills, experience, and freelance work.

## 🌟 Features

* Modern and responsive UI
* Smooth animations and transitions
* Project showcase section
* Skills and technologies section
* Contact section
* Fast performance with Vite
* TypeScript support
* Mobile-friendly design

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* JavaScript
* HTML5 & CSS3

## 📂 Sections

* Home
* About
* Skills
* Projects
* Experience
* Contact

## ⚡ Installation

Clone the repository:

```bash id="it2z6k"
git clone https://github.com/Asyedibrahim/My-Portfolio.git
```

Navigate to the project folder:

```bash id="pn8gk9"
cd portfolio
```

Install dependencies:

```bash id="2iy0to"
npm install
```

Start the development server:

```bash id="z0jvs0"
npm run dev
```

## 📦 Build for Production

```bash id="pwgr5c"
npm run build
```
